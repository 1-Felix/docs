# Setup

Manages Snaplet's configuration.

## `setup auth`

Authenticates the CLI to a Snaplet Cloud account.

```bash >_&nbsp;terminal
snaplet setup auth
```

## `setup completion`

Generates a shell completion script for the CLI.

```bash >_&nbsp;terminal
snaplet setup completion
```

## `setup init`

Launches the interactive wizard to initialize a new Snaplet project.

```bash >_&nbsp;terminal
snaplet setup init
```

## `setup project`

Links a project from Snaplet Cloud.

```bash >_&nbsp;terminal
snaplet setup project
```

## `setup pull`

Pulls a project's configuration from Snaplet Cloud.

```bash >_&nbsp;terminal
snaplet setup pull
```

## `setup push`

Pushes a project's configuration to Snaplet Cloud.

```bash >_&nbsp;terminal
snaplet setup push
```

## `setup types`

Generates TypeScript types for the `snaplet.config.ts` configuration file.

```bash >_&nbsp;terminal
snaplet setup types
```
