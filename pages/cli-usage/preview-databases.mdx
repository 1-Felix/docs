# Preview Databases

Manages preview databases for a project.

## `preview-databases create`

Creates a preview database from a snapshot.

```bash >_&nbsp;terminal
snaplet preview-databases create [snapshot-id]
```

### Arguments

#### `[snapshot-id]`

The identifier of the snapshot to use as the basis for the preview database.

### Flags

#### `--name`
Use a custom name for the preview database.

#### `--git`

Name the preview database after the current git branch.

## `preview-databases drop`

Drops a preview database.

```bash >_&nbsp;terminal
snaplet preview-databases drop [preview-database-id]
```

### Arguments

#### `[preview-database-id]`
The identifier of the preview database to drop.

### Flags

#### `--git`
Name the preview database after the current git branch.

## `preview-databases list`

Lists all preview databases under a project.

```bash >_&nbsp;terminal
snaplet preview-databases list
```

### Flags

#### `--tags`
Apply tag-based filters to the snapshots.

#### `--latest`
Use the most recent snapshot.

## `preview-databases url`

Displays the connection url of a preview database.

```bash >_&nbsp;terminal
snaplet preview-databases url [preview-database-id]
```

### Arguments

#### `[preview-database-id]`
The identifier of the preview database.

### Flags

#### `--git`
Name the preview database after the current git branch.
