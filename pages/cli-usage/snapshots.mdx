# Snapshots

Manages snapshots for a project.

## `snapshots capture`

Captures a snapshot from a source database.

```bash >_&nbsp;terminal
snaplet snapshots capture [source-database-url]
```

### Arguments

#### `[source-database-url]`

The source database connection url to capture from. Defaults to the `SNAPLET_SOURCE_DATABASE_URL` environment variable or the configured source database url in `.snaplet/config.json`.

### Flags

#### `--output`

A custom destination path for the snapshot.

#### `--tags`

Attach tags to the snapshot.

## `snapshots restore`

Restores a snapshot to a target database.

```bash >_&nbsp;terminal
snaplet snapshots restore [snapshot-id | path]
```

### Arguments

#### `[snapshot-id | path]`

Sets the snapshot identifier to restore or its path.

### Flags

#### `--data`

Restores the data. Defaults to `true`.

#### `--drop-database`

Drops the target database. Defaults to `true`.

#### `--exclude-tables`

Excludes the specified tables from being restored.

#### `--latest`

Selects the latest snapshot.

#### `--schema`

Restores the database schema. Defaults to `true`.

#### `--tables`

Restores only the specified tables. Defaults to the tables captured in the snapshot.

#### `--target`

Specifies the target database connection url. Defaults to the `SNAPLET_TARGET_DATABASE_URL` environment variable or the configured target database url in `.snaplet/config.json`.

#### `--tags`

Filters the snapshots by tags.

#### `--yes`

Skips the confirmation prompt if the target database is non local.

## `snapshots share`

Shares a local snapshot by uploading it to Snaplet Cloud.

```bash >_&nbsp;terminal
snaplet snapshots share [snapshot-id | path]
```

### Arguments

#### `[snapshot-id | path]`

Sets the snapshot identifier to share or its path.

### Flags

#### `--latest`

Selects the latest snapshot.

#### `--no-encrypt`

Disables encrypting the snapshot before uploading.

#### `--tags`

Attaches tags to the snapshot.
